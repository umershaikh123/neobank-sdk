Initial release of @raga-neobank/core SDK

- Type-safe SDK for Raga Neobank API
- Support for all 7 API endpoints (vaults, user, portfolio, transactions)
- Custom error handling with NeobankError
- Dual package support (ESM and CommonJS)
- Full TypeScript type definitions
- Zero runtime dependencies (uses native fetch)
- Environment-agnostic (Node.js 18+ and modern browsers)
